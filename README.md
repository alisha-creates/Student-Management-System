# 🎓 Student Management System (Spring Boot CRUD API)

A simple **Student Management System REST API** built using **Spring Boot**.
This project demonstrates basic **CRUD (Create, Read, Update, Delete)** operations using a **3-Layer Architecture (Controller → Service → Repository)**.

The application allows users to manage student records such as adding, viewing, updating, and deleting students.

---

## 🚀 Features

* Add a new student
* View all students
* View student by ID
* Update student details
* Delete a student
* REST API testing using Postman
* Clean 3-layer Spring Boot architecture

---

## 🛠 Technologies Used

* Java
* Spring Boot
* REST API
* Maven
* Postman (for API testing)
